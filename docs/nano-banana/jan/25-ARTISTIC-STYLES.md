# VibeDeck 25 Artistic Style Variations

**Date**: 2026-01-14
**Goal**: Generate 25 wildly different DEVICE DESIGNS (not backgrounds)
**Budget**: Use cheap models (~$0.02-0.04 per image)

---

## Base Device Description (Use in ALL prompts)

```
A handheld hardware controller device with:
- Small LCD screen at top showing text
- Large central rotary knob/dial with knurled metal texture
- 6 context buttons (CTX 1-6) arranged around the knob
- Bottom row of action buttons (YES, NO, OTHER, microphone, SEND, CHAT)
- Colorful LED backlighting under buttons
- Coiled cable attached
```
